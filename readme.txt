6. 作品のゲーム部分、プログラム部分のアピールポイント
   【ゲーム部分のアピールポイント】
   ・「ジャスト回避」による手に汗握るスリリングな駆け引きと爽快なカウンターアクション
     敵の激しい弾幕を引きつけてタイミングよく回避（スペースキー）することで「JUST EVASION!」を発動できます。発動時はド派手な放射状ブラーエフェクト（RadialBlur）の演出とともにプレイヤーが一時無敵となり、画面上の敵の弾幕をすべて消し去ります。さらに反撃として、敵へ追従・追撃する12発のカウンター誘導弾を一斉掃射し、防戦一方から一瞬で攻勢に転じる爽快感を演出しました。
   
   ・プレイヤーを引き込むタイトル画面の「オートデモプレイ」
     タイトル画面のまま一定時間操作がないと、自動的にデモ用のステージCSVが読み込まれ、AIによる自動操作でゲームが動き始めます。これにより、ゲーム開始前からプレイヤーにゲーム体験を視覚的にアピールできるようにしました。

   【プログラム部分のアピールポイント】
   ・快適なゲームプレイを提供する「誘導弾（ホーミング）の近接アシストアルゴリズム」
     ジャスト回避カウンター弾（PlayerCircusBullet）は、単に敵へ一直線に向かうのではなく、サイン波のうねり（カオス軌道）を伴いながら美しく飛行します。さらに、敵との距離が近づいた（6.0f以内）際には「うねりの振幅を徐々にフェードアウトさせて直線軌道にしつつ、旋回力（Turn Speed）を最大2倍まで動的に向上させる」アシスト処理を実装しました。これにより、誘導弾が敵の至近距離で曲がりきれずに外れる（旋回ループに陥る）問題を完全に防ぎ、確実にヒットする非常に手触りの良いゲームフィールを実現しました。
   
   ・DirectX 12の特性を活かしたパーティクル最適化
     弾幕シューティングアクションとしての描画負荷に耐えられるよう、オブジェクトプールを用いたパーティクル管理機構を設計しました。移動時のトレイル（軌跡）、ブースターの火花、衝突時の爆発などで一度に数千個のパーティクルを画面上に放出しても処理落ちせず、安定して60FPSで動作するように最適化しています。
   
   ・拡張性に優れたステートパターン設計
     ゲームシーン遷移だけでなく、ステージ内部の状態遷移（Ready, Playing, Pause, GameOver, GameClear, Tutorial）についても `IStageState` インターフェースを用いたステートパターンを構築しました。これにより、各状態の処理がクラスとしてきれいに分離され、新しい状態の追加やデバッグが極めて容易な疎結合設計に仕上げています。
