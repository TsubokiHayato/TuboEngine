Dodgeリング（クールダウンゲージ）を「スプライトのみ」で時計回りに減らす表現

- 方式: 細い矩形セグメント画像を複数枚(Sprite)回転配置して円弧（リング）を作る。
- 必要テクスチャ: UI/seg_white.png
  - 透過PNG
  - 中央に白い長方形（例: 64x16）
  - 周囲は完全透明
  - 角は少し丸めると見栄えが良い

- 表現:
  - dodgeReadyRatio=1.0 の時は全セグメント表示（リングが満タン）。
  - 0.0 へ近づくにつれ、セグメントを右回りに非表示にしていく。
  - 12時を開始点にする（StartAngle=-90deg）。

- 色:
  - HPとは別系統（シアン/青系）推奨。
